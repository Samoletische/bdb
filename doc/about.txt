devideForAll - равномерное списание с каждого депозита активных пользователей. В значении true есть возможность показать сколько списано с каждого депозита.
purseCommon - определяет наличие депозитов у каждого пользователя внутри кошелька. Значение true - означает отсутствие депозитов.
purseProtect - в случае наличия депозитов определяет показывать ли депозиты других участников кошелька.
